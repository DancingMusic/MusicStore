Getting Started

MusicStoreSdk provides the API client, data models, and type definitions for the Music store — covering track listing, purchasing, and order verification.

Install

npm install @dancingmusic/music-store-sdk

Quick start

import { MusicStoreClient } from '@dancingmusic/music-store-sdk';

const client = new MusicStoreClient({
  baseUrl: 'https://api.dancingmusic.dev/music',
  token: 'your-auth-token',
});

// List all tracks
const tracks = await client.list({ genre: 'electronic' });

// Purchase a track
const order = await client.createOrder(tracks[0].id);

Configuration

The client accepts a MusicStoreClientOptions object:

OptionTypeDescription
baseUrlstringMusic store API base URL (trailing slash is trimmed)
tokenstringAuth token for authenticated endpoints optional
timeoutMsnumberRequest timeout in milliseconds optional

MusicStoreClient

The main API client class for interacting with the music store.

MethodSignatureDescription
list(query?: MusicListQuery) => Promise<MusicTrack[]>List tracks with optional filters
get(trackId: string) => Promise<MusicTrack | null>Get a single track by ID
createOrder(trackId: string) => Promise<MusicOrder>Create a purchase order
verifyOrder(orderId: string) => Promise<MusicOrder | null>Verify order payment status
config{ baseUrl, hasToken }Read-only resolved configuration

Error handling

Methods throw a standard Error when required parameters are missing (e.g. empty trackId). Network errors propagate as-is from the underlying fetch implementation.

MusicTrack

Represents a music track in the store.

FieldTypeDescription
idstringUnique track identifier
titlestringTrack title
artiststringArtist name
albumstringAlbum name optional
durationSecnumberDuration in seconds
pricenumberPrice amount
currencystringCurrency code (e.g. "USD")
versionstringTrack version
createdAtstringISO creation timestamp
updatedAtstringISO update timestamp

MusicOrder

Represents a purchase order.

FieldTypeDescription
orderIdstringUnique order identifier
trackIdstringPurchased track ID
amountnumberPayment amount
currencystringCurrency code
status'pending' | 'paid' | 'failed' | 'refunded'Order status

MusicListQuery

Query parameters for the list() method.

FieldTypeDescription
pagenumberPage number optional
pageSizenumberItems per page optional
keywordstringSearch keyword optional
genrestringFilter by genre optional
