Developer

五分钟上手

安装依赖，编写几行代码，即可接入音乐数据。

Install
# Install MusicStore SDK npm install @dancingmusic/music-store # Install MusicConnect (data source connector) npm install @dancingmusic/music-connect # Install NetEase Cloud Music connector npm install @dancingmusic/music-connect-netease
Quick Start
import { MusicStore } from '@dancingmusic/music-store'; import { MusicConnect } from '@dancingmusic/music-connect'; import { NeteaseConnector } from '@dancingmusic/music-connect-netease'; // Register data source const connect = new MusicConnect(); connect.register(new NeteaseConnector()); // Create store instance const store = new MusicStore({ connect }); // Search across all sources const results = await store.search('晴天'); console.log(results);
